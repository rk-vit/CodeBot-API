API Documentation

This API provides endpoints for generating and modifying HTML content based on user inputs. Powered by Gemini AI to process requests and return HTML code snippets.

Base URL

http://localhost:3000

Endpoints

1. POST /theme/:theme/:content

Description: Generates an HTML page based on the provided theme and content.

Request:

Use path parameters to pass the HTML theme and content.

POST /theme/{theme}/{content}

Example:

POST /theme/<html><head><title>Theme</title></head><body></body></html>/Welcome to my website!

Response:

Status Code: 200 OK
Content-Type: application/json
{
    "code": "<html><head><title>Theme</title></head><body>Welcome to my website!</body></html>"
}

Error Response:

Status Code: 500 Internal Server Error
Content-Type: application/json
{
    "error": "An error occurred while processing your request."
}

2. POST /changes/:code/:change

Description: Modifies existing code based on the provided changes.

Request:

Use path parameters to pass the code and the changes you want to make.

POST /changes/{code}/{change}

Example:

POST /changes/<html><head><title>Old Title</title></head><body>Old Content</body></html>/Update the title to 'New Title' and the content to 'New Content'.

Response:

Status Code: 200 OK
Content-Type: application/json
{
    "code": "<html><head><title>New Title</title></head><body>New Content</body></html>"
}

Error Response:

Status Code: 500 Internal Server Error
Content-Type: application/json
{
    "error": "An error occurred while processing your request."
}

Error Codes

  • 400 Bad Request: The request could not be understood by the server due to malformed syntax.
  • 404 Not Found: The requested resource could not be found.
  • 500 Internal Server Error: The server encountered an unexpected condition which prevented it from fulfilling the request.
